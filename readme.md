
### Comment lancer ces fichiers ?

1. **Prérequis :** Assure-toi d'avoir installé les librairies. Ouvre ton terminal et tape :
```bash
pip install pandas matplotlib seaborn streamlit plotly

```


2. **Pour le Notebook :**
* Lance Jupyter Notebook ou Jupyter Lab.
* Crée un nouveau notebook (`.ipynb`).
* Colle le code dans une cellule et exécute.


3. **Pour l'application Streamlit :**
* Crée un fichier texte nommé `app.py`.
* Colle le code ci-dessus dedans.
* Dans ton terminal, navigue vers le dossier où se trouve le fichier et tape :
```bash
streamlit run app.py

```
